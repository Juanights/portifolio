# Portfólio de Dados Interativo com Streamlit

Este projeto é um portfólio profissional desenvolvido em Python utilizando o framework **Streamlit**. Ele foi projetado para ser uma vitrine interativa de projetos de Análise e Ciência de Dados.

## Estrutura do Portfólio

1.  **🏠 Home:** Boas-vindas, resumo profissional e links rápidos.
2.  **👤 Sobre Mim:** Trajetória acadêmica, habilidades técnicas (hard skills) e ferramentas.
3.  **📊 Projetos:** Galeria de projetos interativos.
    *   *Análise Exploratória (EDA):* Visualizações dinâmicas de um dataset.
    *   *Machine Learning:* Um modelo preditivo onde o usuário pode inserir dados e ver a previsão.
    *   *Dashboard de BI:* Gráficos de indicadores de desempenho.
4.  **✉️ Contato:** Links para LinkedIn, GitHub e formulário de contato.

## Tecnologias Utilizadas

*   **Python:** Linguagem base.
*   **Streamlit:** Framework para criação da interface web.
*   **Pandas:** Manipulação de dados.
*   **Plotly / Matplotlib:** Visualizações interativas.
*   **Scikit-learn:** Modelagem de Machine Learning.
